# Reports Folder

This folder contains business reports and dashboards for the Customer Churn Analysis project.

---

## 📁 Files

| File | Description | Audience |
|------|-------------|----------|
| `executive_summary.md` | High-level business report with findings and recommendations | Management, Stakeholders |
| `churn_dashboard.xlsx` | Interactive Excel dashboard with slicers and PivotCharts | Stakeholders, Analysts |

---

## 📊 Excel Dashboard Features

The Excel dashboard (`churn_dashboard.xlsx`) includes:
- **KPIs**: Total Customers, Churn Rate, Avg Tenure, Avg Monthly Charges
- **Charts**: Churn by Contract, Tenure, Internet Service, Payment Method
- **Slicers**: Filter by Contract, Internet, Payment Method, Senior Citizen
- **Interactivity**: All charts update instantly when slicers are clicked
