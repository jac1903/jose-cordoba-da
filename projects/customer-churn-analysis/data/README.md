# Data Folder

This folder contains all datasets used for the Customer Churn Prediction & Analysis project.

---

## 📁 Data Files

| File | Description | Source |
|------|-------------|--------|
| `telco_churn_raw.csv` | Original Telco Customer Churn dataset (7,043 rows) | IBM / Kaggle |
| `customers.csv` | Demographics table (customer_id, gender, senior_citizen, partner, dependents) | Split from raw data |
| `accounts.csv` | Account & billing table (customer_id, tenure, contract, payment_method, monthly_charges, total_charges, churn) | Split from raw data |
| `services.csv` | Services table (customer_id, phone_service, internet_service, online_security, tech_support, ...) | Split from raw data |
| `full_churn_data_for_excel.csv` | Combined dataset with `tenure_group` and `churn_probability`, exported for Excel dashboard | Generated by Python |

---

## 📊 Relational Database Design

The raw CSV was normalized into **3 relational tables** to practice database design and reduce redundancy:

| Table | Primary Key | Foreign Key |
|-------|-------------|-------------|
| `customers` | `customer_id` | - |
| `accounts` | `customer_id` | `customer_id` → `customers` |
| `services` | `customer_id` | `customer_id` → `customers` |
