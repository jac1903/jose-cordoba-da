\# Data Folder



This folder contains all datasets used for the Customer Churn Prediction \& Analysis project.



\---



\## 📁 Data Files



| File | Description | Source |

|------|-------------|--------|

| `telco\_churn\_raw.csv` | Original Telco Customer Churn dataset (7,043 rows) | IBM / Kaggle |

| `customers.csv` | Demographics table (customer\_id, gender, senior\_citizen, partner, dependents) | Split from raw data |

| `accounts.csv` | Account \& billing table (customer\_id, tenure, contract, payment\_method, monthly\_charges, total\_charges, churn) | Split from raw data |

| `services.csv` | Services table (customer\_id, phone\_service, internet\_service, online\_security, tech\_support, ...) | Split from raw data |

| `full\_churn\_data\_for\_excel.csv` | Combined dataset with `tenure\_group` and `churn\_probability`, exported for Excel dashboard | Generated by Python |



\---



\## 📊 Relational Database Design



The raw CSV was normalized into \*\*3 relational tables\*\* to practice database design and reduce redundancy:



| Table | Primary Key | Foreign Key |

|-------|-------------|-------------|

| `customers` | `customer\_id` | - |

| `accounts` | `customer\_id` | `customer\_id` → `customers` |

| `services` | `customer\_id` | `customer\_id` → `customers` |
