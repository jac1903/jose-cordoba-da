# SQL Queries for E-commerce Analysis

This folder contains SQL queries used for the RFM (Recency, Frequency, Monetary) analysis.

## Files
- `customer_segmentation.sql` — Main RFM segmentation query

## How to Run
1. Open MySQL Workbench.
2. Connect to your database.
3. Copy and paste the SQL code.
4. Run the query to see the customer segments.
