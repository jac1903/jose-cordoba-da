# Data Folder

This folder contains the dataset used for the analysis.

## Dataset Source
Brazilian E-commerce Dataset (Olist) from Kaggle

## How to Download
1. Visit: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
2. Download the CSV files.
3. Place them in this folder.

## Files Used
- `olist_orders_dataset.csv` — Main orders table
- `olist_order_items_dataset.csv` — Order items with prices
- `olist_customers_dataset.csv` — Customer information

## Size
The full dataset is ~100,000+ rows. A sample file is included for quick preview.
