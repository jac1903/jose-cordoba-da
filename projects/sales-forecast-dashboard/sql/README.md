# SQL Queries for Sales Performance Dashboard

This folder contains SQL queries used for the sales analysis and dashboard.

## Files
- `sales_metrics.sql` — Complete set of queries for sales analysis

## Queries Included

| # | Query | Purpose |
|---|-------|---------|
| 1 | Preview data | Verify data loaded correctly |
| 2 | Overall summary | Key KPIs at a glance |
| 3 | Sales by Region | Regional performance comparison |
| 4 | Monthly trend | Time series for forecasting |
| 5 | YoY growth | Year-over-year performance |
| 6 | Top products | Best-selling products |
| 7 | Category performance | Category-level analysis |
| 8 | Customer segment | Segment analysis |
| 9 | Quarterly regional | Quarterly breakdown |
| 10 | MoM growth | Month-over-month trends |
| 11 | Dashboard query | Quick KPIs for visualization |

## How to Run
1. Open MySQL Workbench.
2. Connect to the `sales_analysis` database.
3. Copy and paste the desired query.
4. Run the query to see results.
